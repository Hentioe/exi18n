# ExI18n

**ExI18n** is key-based internationalization library for Elixir.

## Installation

Add `exi18n` to your list of dependencies and to `applications` in `mix.exs`:

```elixir
def deps do
  [
    {:exi18n, "~> 0.1.0"}
  ]
end

def application do
  [applications: [:exi18n]]
end
```
